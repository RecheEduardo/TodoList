# Funcionalidades:
<p align="justify"> No site, é possível criar uma tabela que exibe as suas tarefas, podendo variar entre <strong>Pendente</strong> e <Strong>Concluída</Strong>, gerando dinamismo e melhor espaço para administrá-las e contribuir para um layout limpo e organizado</p>
<p align="justify"> Dentro das funcionalidades, vale destacar a função de <strong>Filtrar</strong>, que ordena as tarefas conforme a opção de visualização do usuário. Além disso, vale ressaltar a aba de <strong>Pesquisa</strong>, que possibilita o usuário procurar uma tarefa específica da sua lista, gerando assertividade para consultar suas pendências.</p>
